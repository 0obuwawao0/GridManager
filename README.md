# listManager.js
###实现功能
listManager.js可快速的对table标签进行实例化，实例化后将实现以下功能:

- 表格宽度调整、位置更换、列展示隐藏、表头吸顶等前端交互效果
- 表格各项单项排序或组合排序功能
- 表格ajax分页功能
- 表格交互效果记忆功能,如记忆用户的列宽、列位置、列可视状态、每页显示条数
- 配置参数支持简易、标准两种模式,并在各callback中返回与配置相符模式的参数
